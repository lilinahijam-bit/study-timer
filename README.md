# Study Cooking Timer 🍳

A simple productivity web app that tracks study time like cooking.

## Features
- ⏱️ Timer system
- 🍳 Cooking-style feedback (undercooked / perfect / overcooked)
- 📊 Study progress chart
- 💾 Data saved using local storage

## Tech Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
Open index.html in browser
