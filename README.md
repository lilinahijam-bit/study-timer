# 🍳 Study Cooking Timer

A fun productivity web app that tracks study time like cooking.

## 🚀 Features
- ⏱️ Real-time study timer
- 🍞 Undercooked / 🍳 Perfect / 🔥 Overcooked feedback
- 🎯 Score system
- 📊 Study progress chart
- 💾 Data stored using localStorage

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
